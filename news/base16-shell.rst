**Added:**

* Add alias to `base16 shell`_

* Installation / Usage
    1. To install use pip

    .. code-block:: bash

        python3 -m pip install xontrib-base16-shell

    2. Add on ``~/.xonshrc``

    .. code:: python
        :number-lines:

        $BASE16_SHELL = $HOME + "/.config/base16-shell/"
        xontrib load base16_shell
